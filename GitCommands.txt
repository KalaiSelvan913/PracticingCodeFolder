Git Command Notes

  -> "git init" for initalizing the table

  -> "git remote add origin <url of git repo> for adding the repo to local directory

  -> "git status" for viewing the untracked files and committed files

  -> "git add ." to add the file to stage area

  -> "git commit -m "add <repo name> master"" for commiting the local repo 

  -> "git push orgin master" for pushing the local repot to git hub
